# React/Redux-Blog

###Getting Started###

There are two methods for getting started with this repo.

####Familiar with Git?#####
Checkout this repo, install dependencies, then start the gulp process with the following:

```
	> git clone github.com/walifaizy/React-Redux-Blog.git
	> cd React-Redux-Blog
	> npm install
	> npm start
```


```
	> npm install
	> npm start
```
